# Birthday Checklist

Мобильное PWA-приложение для локального checklist закупок на день рождения.

## Что внутри

- Группированный список ингредиентов и базовых продуктов.
- Отметки купленных позиций сохраняются в `localStorage`.
- Нет backend, регистрации, общей базы данных или синхронизации между устройствами.
- Фильтры: `Все`, `Осталось`, `Куплено`.
- Кнопка сброса отметок и кнопка поделиться/скопировать оставшийся список.

## Локальный запуск

Можно открыть `index.html` напрямую в браузере. Для проверки PWA и service worker лучше запустить простой static server:

```bash
python3 -m http.server 4173
```

После этого открыть:

```text
http://localhost:4173/
```
