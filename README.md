# Birthday Checklist

Мобильное PWA-приложение для локального checklist закупок и плана готовки на день рождения.

## Что внутри

- Группированный список ингредиентов и базовых продуктов.
- Режим `Готовка` с поэтапным таймлайном, ролями, ингредиентами и шагами приготовления.
- Отметки купленных позиций и готовых этапов сохраняются в `localStorage`.
- Нет backend, регистрации, общей базы данных или синхронизации между устройствами.
- Контекстные фильтры: `Все`, `Осталось`, `Куплено` для покупок и `Все`, `Осталось`, `Готово` для готовки.
- Кнопка сброса отметок и кнопка поделиться/скопировать оставшийся список или план готовки.

## Локальный запуск

Можно открыть `index.html` напрямую в браузере. Для проверки PWA и service worker лучше запустить простой static server:

```bash
python3 -m http.server 4173
```

После этого открыть:

```text
http://localhost:4173/
```
